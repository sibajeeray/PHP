5 phases of building a website
---------------------------------
    1.Discovery -> Inspiration and decide features for the dream site.
    2.Design -> Creating WireFrames and then ful Design mockups.
    3.Code/Development -> Implementing the designs and features.
    4.QA -> Testing and quality assurance phase.
    5.Deployment -> Moving code to live server.
